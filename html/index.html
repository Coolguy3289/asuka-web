About Asuka

Asuka is a Discord bot to add Slack like entry control. This bot will allow you to set a domain to which users entering your server will have to prove they possess an email matching. After which, they will be allowed to browse the server like any other user.

Features

Moderation

The bot will have to get a verification through email to show that it is actually a user that joins and not a fake user. This prevents from a lot of spamming by fake users.

Web Dashboard

A configurable web dashboard. You can easily add the bot through here. You can do the rest on your discord server.


Getting Started

To get started with Asuka, login with discord and select the server that you would like to setup.
